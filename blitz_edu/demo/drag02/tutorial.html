Ada banyak cara untuk menggunkan Blitz Edu. salah satunya adalah dengan mendownload template dan mengeditnya.

Anda bisa mendownload template disini: [template.js]

Kode minimal dari BE adalah sebagai berikut: 

Grafis(240, 320);

let spr = Muat("./gbr/box.png", true);
Posisi(spr, 160, 120);

function Loop() {
    Bersih();
    Gambar(spr);
}

Penjelasan tiap baris adalah sebagai berikut:

Grafis(240, 320);

Ini adalah pintu masuk dari BE. Fungsi ini harus dipanggil di awal, untuk mengeset kanvas.
BE bekerja di kanvas dan butuh kanvas untuk bisa berfungsi.
Fungsi ini memiliki dua parameter utama: panjang dan lebar.
panjang dan lebar menentukan panjang dan lebar kanvas.
Parameter sisanya bisa Anda baca di sini

let spr = Muat("./gbr/box.png", true);

Fungsi Muat() berfungsi untuk memuat gambar dan di konvert ke dalam bentuk Sprite.
Sprite adalah tokoh utama dari BE. Tanpa Sprite maka tidak akan ada yang ditampilkan di layar.
Kita memuat Sprite dari URL, bisa lokal ataupun online.
Bila Anda meload gambar dari online, pastikan Anda mendapatkan ijin dulu. Parameter kedua dari fungsi ini adalah untuk menentukan apakah Sprite bisa di drag atau tidak.
Drag-Drop adalah fitur bawaan dari Sprite.
BE fokus pada interaksi, dan sepertinya drag-drop adalah feature yang bakal banyak dipakai. Maka dari itulah Saya memutuskan bahwa drag-drop harus ada secara default.

Posisi(spr, 160, 120);

Fungsi ini untuk menentukan posisi Sprite. 

function Loop() {
    Bersih();
    Gambar(spr);
}

Fungsi Loop() adalah fungsi untuk perulangan, akan dipanggil 60 kali per detiknya, atau 60 fps, tergantung kecepatan device. Dalam fungsi ini kita memanggil dua fungsi lainnya:

Bersih()

Sesuai namanya, maka fungsi ini akan membersihkan layar.

Gambar(spr)

Sesuai namanya juga, fungsi ini akan menggambar di layar. Parameternya adalah Sprite yang akan digambar.
